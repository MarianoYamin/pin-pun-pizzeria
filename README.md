# Pin Pun · Sitio web

Sitio editorial multi-página para la pizzería Pin Pun, fundada en 1927 en Av. Corrientes 3954, Almagro (CABA).

## Stack

- **Astro 4** (estático, multi-página)
- **TypeScript strict**
- **GSAP + ScrollTrigger** + **Lenis** para microanimaciones
- **CSS** nativo con custom properties

## Comandos

```bash
npm install         # instalar dependencias
npm run dev         # servidor de desarrollo en http://localhost:4321
npm run build       # build estático a /dist
npm run preview     # preview del build
```
